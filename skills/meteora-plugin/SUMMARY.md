# meteora-plugin
Meteora DLMM plugin for searching pools, getting swap quotes, checking positions, executing swaps, and adding liquidity on Solana.

## Highlights
- Search and filter Meteora DLMM liquidity pools by TVL, APY, volume and token pairs
- Get real-time swap quotes via DEX aggregator integration
- View user LP positions with on-chain token amount calculations
- Execute token swaps with price impact warnings and slippage protection
- Add liquidity using SpotBalanced strategy with 70-bin position management
- Remove liquidity with optional position closure to reclaim rent
- Quickstart command for personalized onboarding based on wallet balances
- Direct integration with onchainos CLI for secure transaction execution
