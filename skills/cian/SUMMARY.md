# cian
CIAN Yield Layer provides multi-chain ERC4626 yield vaults for ETH/BTC LST assets across Ethereum, Arbitrum, BSC, and Mantle networks.

## Highlights
- Multi-chain yield aggregator with ~$500M+ TVL across 4 networks
- Delta-neutral automated strategies for ETH and BTC derivative assets
- ERC4626-compliant vaults supporting stETH, rsETH, pumpBTC, and other LSTs
- Non-instant withdrawal system with rebalancer queue processing
- Support for multiple token inputs via optionalDeposit functionality
- Real-time APY tracking and TVL monitoring across all vaults
- Transparent vault proxy architecture with upgrade safety
- Cross-chain position management and earnings tracking
