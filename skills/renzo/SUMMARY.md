# renzo
Renzo EigenLayer liquid restaking — deposit ETH or stETH to receive ezETH and earn restaking rewards.

## Highlights
- Deposit native ETH to mint ezETH liquid restaking tokens
- Deposit stETH to mint ezETH with approve + deposit flow
- Earn EigenLayer AVS restaking rewards automatically
- Check current restaking APR from Renzo protocol
- Query ezETH and stETH balances for any address
- View total value locked (TVL) in Renzo protocol
- Non-rebasing ezETH token with appreciating exchange rate
- Ethereum mainnet integration with TEE-sandboxed signing
