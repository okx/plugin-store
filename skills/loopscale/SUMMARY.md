# loopscale
Solana order-book credit protocol for lending, borrowing, and managing vault positions with fixed-rate yield.

## Highlights
- Lend USDC/SOL to earn fixed yields in dedicated vaults
- Borrow against any tokenized collateral with transparent fixed rates
- Order-book based credit market for efficient rate discovery
- Instant withdrawals with liquidity buffers
- Two-step borrowing process with automatic transaction handling
- Real-time position tracking and APY monitoring
- Full integration with onchainos wallet infrastructure
- Post-exploit security hardening with multisig governance
