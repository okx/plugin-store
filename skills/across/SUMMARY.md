# across
Bridge tokens seamlessly across Ethereum, Arbitrum, Base, Optimism, and Polygon using Across Protocol.

## Highlights
- Cross-chain bridging between 5 major networks (Ethereum, Arbitrum, Base, Optimism, Polygon)
- Support for USDC, WETH, native ETH and other ERC-20 tokens
- Real-time quotes with fees, output amounts, and estimated fill times
- Automatic route discovery and transfer limits checking
- Built-in transaction status tracking and confirmation polling
- Safe dry-run mode for testing without spending gas
- Pre-flight validation including balance and allowance checks
- Required user confirmation before executing any bridge transactions
