# flap
Token launchpad on BSC for creating and trading tokens with bonding curves via Flap Protocol.

## Highlights
- Create standard or tax tokens on BSC with bonding curves
- Buy and sell tokens directly through Flap Protocol bonding curves
- Automatic CREATE2 salt grinding for vanity token addresses (8888/7777 suffixes)
- Support for tax tokens with customizable buy/sell tax rates
- Query bonding curve state including price, reserves, and graduation progress
- Automatic DEX graduation detection at 16 BNB threshold
- Built-in slippage protection and sell tax warnings
- Direct integration with BSC mainnet and onchainos wallet
