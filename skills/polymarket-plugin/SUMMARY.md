# polymarket-plugin

Trade prediction markets on Polymarket — buy YES/NO outcome tokens, manage positions, and redeem winnings on Polygon with support for gasless proxy trading.

## Highlights
- Trade binary (YES/NO) and categorical prediction markets on Polymarket
- Support for both EOA mode (direct wallet) and POLY_PROXY mode (gasless trading)
- Browse markets by category (sports, elections, crypto) or trending/breaking events
- Specialized 5-minute crypto up/down markets for short-term trading
- Automatic order signing and settlement via Polymarket's CLOB
- Position tracking and automatic winning token redemption
- Region verification to ensure compliance with Polymarket's terms
- One-time proxy wallet setup for gas-free trading experience
