# curve
Curve DEX plugin for swapping stablecoins, adding/removing liquidity, and querying pool data across multiple chains.

## Highlights
- Swap stablecoins on Curve Finance across Ethereum, Arbitrum, Base, Polygon, and BSC
- Add and remove liquidity from Curve pools with automatic approval handling
- Query pool information including APY, TVL, and fees
- Get real-time swap quotes with slippage protection
- Check LP token balances across all supported chains
- Support for both proportional and single-coin liquidity withdrawals
- Built-in dry-run mode for previewing transactions before execution
- Automatic token approval management with confirmation polling
