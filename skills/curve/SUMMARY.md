# curve
Curve DEX plugin for swapping stablecoins and managing liquidity on Curve Finance across multiple chains.

## Highlights
- Swap stablecoins on Curve Finance across Ethereum, Arbitrum, Base, Polygon, and BSC
- Add and remove liquidity from Curve pools with automatic token approvals
- Query pool information, APY rates, and TVL data from official Curve API
- Check LP token balances and estimated underlying values
- Get real-time swap quotes with slippage protection
- Preview all transactions with dry-run mode before execution
- Automatic router detection and multi-chain support
- Built-in security checks and confirmation prompts for write operations
