
# curve -- Skill Summary

## Overview
The Curve plugin provides comprehensive access to Curve Finance, a leading DEX specialized in stablecoin trading with minimal slippage. It enables users to swap tokens, provide liquidity to earn fees and CRV rewards, and manage LP positions across Ethereum, Arbitrum, Base, Polygon, and BSC networks.

## Usage
Use natural language triggers like "swap on Curve," "add liquidity to Curve pool," or "show Curve pools" to interact with the protocol. The plugin automatically handles token approvals and provides transaction previews before execution.

## Commands
| Command | Description |
|---------|-------------|
| `get-pools` | List available Curve pools with APY and TVL data |
| `get-pool-info` | Get detailed information about a specific pool |
| `get-balances` | Show LP token balances for connected wallet |
| `quote` | Get swap quotes with price impact calculation |
| `swap` | Execute token swaps on Curve pools |
| `add-liquidity` | Add liquidity to pools (single or multi-coin) |
| `remove-liquidity` | Withdraw liquidity proportionally or as single coin |

## Triggers
Activate this skill when users want to trade stablecoins with minimal slippage, earn yield through liquidity provision on Curve Finance, or need to check Curve pool information and APY rates. Use for Curve-specific operations only, not general DEX activities.
