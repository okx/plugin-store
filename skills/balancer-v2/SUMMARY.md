# balancer-v2
A Plugin Store skill for interacting with Balancer V2 multi-token AMM DEX on Ethereum and Arbitrum.

## Highlights
- Swap tokens via Balancer's unified Vault entry point
- Query top pools by liquidity and get detailed pool information
- Get on-chain swap quotes using BalancerQueries contract
- Add and remove liquidity through joinPool and exitPool operations
- View LP positions and BPT token holdings
- Support for Arbitrum (primary) and Ethereum chains
- Unified architecture through single Vault contract address
- Built-in slippage protection and transaction simulation
