# meteora
Meteora DLMM plugin for Solana — search liquidity pools, get swap quotes, view user positions, and execute token swaps.

## Highlights
- Search and filter Meteora DLMM pools by TVL, APY, volume, and fee ratios
- Get detailed pool information including configuration, reserves, and fee structure
- Obtain swap quotes using onchainos DEX aggregator on Solana
- View user LP positions with token amounts, bin ranges, and unclaimed fees
- Execute token swaps with dry-run preview and user confirmation
- Built-in risk warnings for high price impact and APY pools
- Support for major Solana tokens (SOL, USDC, USDT)
- Direct REST API integration for fast read operations
