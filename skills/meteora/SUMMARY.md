# meteora
A Meteora DLMM plugin for Solana that enables searching pools, getting swap quotes, checking positions, and executing token swaps.

## Highlights
- Search and filter Meteora DLMM liquidity pools by TVL, APY, volume, and fee ratios
- Get real-time swap quotes using onchainos DEX aggregator on Solana
- View user LP positions including token amounts, bin ranges, and unclaimed fees
- Execute token swaps with dry-run preview and user confirmation workflow
- Built-in risk warnings for high price impact (>5%) and high-yield pools (>50% APY)
- Support for popular Solana tokens (SOL, USDC, USDT) with mint address lookup
- Direct REST API integration for read operations, no wallet required
- Comprehensive pool details including configuration, TVL, and fee structure
