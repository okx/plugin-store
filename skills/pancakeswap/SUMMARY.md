# pancakeswap
Swap tokens and manage concentrated liquidity on PancakeSwap V3 across BNB Chain and Base networks.

## Highlights
- Token swapping via PancakeSwap V3 SmartRouter with best price discovery across all fee tiers
- Concentrated liquidity position management (mint, remove, view positions)
- Multi-chain support for BNB Chain (BSC) and Base networks
- Real-time swap quotes without transaction costs using QuoterV2
- Complete LP position lifecycle management with NonfungiblePositionManager
- Pool discovery and liquidity analysis across different fee tiers
- Built-in slippage protection and dry-run capabilities for safety
- Automatic token approval handling for seamless user experience
