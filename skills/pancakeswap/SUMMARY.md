# pancakeswap
Swap tokens and manage concentrated liquidity on PancakeSwap V3 on BNB Chain and Base.

## Highlights
- Token swaps via SmartRouter with automatic best price discovery across fee tiers
- Concentrated liquidity provision with precise tick range control
- Support for BNB Chain (BSC) and Base networks
- Real-time quote fetching without gas costs
- LP position management (mint, view, remove liquidity)
- Pool discovery across all fee tiers (0.01%, 0.05%, 0.25%, 1%)
- Automatic slippage protection and multi-step transaction safety
- Integration with onchainos wallet for secure transaction execution
