# clanker
Deploy and manage Clanker ERC-20 tokens on Base and Arbitrum with token launch, search, and LP fee claiming capabilities.

## Highlights
- Deploy new ERC-20 tokens on Base and Arbitrum via Clanker protocol
- Search tokens by creator address or Farcaster username
- List recently deployed Clanker tokens with pagination
- Get real-time token metadata and pricing information
- Claim LP fee rewards for token creators
- Support for custom token vaults with lockup periods
- Built-in security scanning for token interactions
- Seamless integration with onchainos wallet system
