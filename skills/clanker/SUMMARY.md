# clanker
Deploy and manage Clanker ERC-20 tokens on Base and Arbitrum — launch tokens, search by creator, and claim LP fee rewards.

## Highlights
- Deploy new ERC-20 tokens via Clanker protocol on Base and Arbitrum
- Search tokens by creator address or Farcaster username
- List latest token launches with pagination and sorting
- Get real-time token info including price and market cap
- Claim LP fee rewards for token creators
- Built-in security scanning for token safety
- Support for vault lockup and percentage settings
- Integration with OKX OnChainOS wallet and Uniswap V4
