# clanker
Deploy and manage Clanker ERC-20 tokens on Base and Arbitrum — launch tokens, search by creator, and claim LP fee rewards.

## Highlights
- Deploy ERC-20 tokens via Clanker protocol on Base and Arbitrum
- Search tokens by creator address or Farcaster username
- List latest token launches with pagination
- Get real-time token info including price and market cap
- Claim LP fee rewards for deployed tokens
- Security scanning for token safety validation
- Dry-run mode for previewing deployments and claims
- Vault locking options for token supply management
