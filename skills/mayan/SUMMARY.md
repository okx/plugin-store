# mayan
Cross-chain swap plugin supporting Swift/MCTP routes between Solana, Ethereum, Arbitrum, Base, Optimism, Polygon, BSC, and Avalanche.

## Highlights
- Swift route cross-chain swaps (~15 seconds completion)
- MCTP stablecoin-optimized routing
- Support for 8 major chains including Solana and EVM networks
- Automatic route selection (Swift > MCTP > Wormhole)
- ERC-20 token approval handling for EVM chains
- Real-time swap status tracking via Mayan Explorer API
- Native token support (SOL, ETH) and wrapped tokens
- Configurable slippage protection (default 1%, max 3%)
