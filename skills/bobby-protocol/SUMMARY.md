# bobby-protocol
Adversarial AI trading intelligence on OKX X Layer. Three agents debate every trade, a Judge Mode audits quality, and anyone can stake OKB to challenge decisions on-chain. 15 MCP tools (10 free, 5 premium via x402).

## Highlights
- 3-agent adversarial debate: Alpha Hunter proposes, Red Team attacks, CIO decides
- Judge Mode: independent 6-dimension audit (data integrity, adversarial quality, decision logic, risk management, calibration, novelty)
- Adversarial bounties: stake OKB on-chain to prove Bobby was wrong
- 15 MCP tools over Streamable HTTP with x402 payment protocol
- 4 verified smart contracts on X Layer mainnet
- On-chain conviction oracle and commit-reveal track record
- Non-custodial: all bounty tools return unsigned calldata
- Uses OKX OnchainOS APIs for smart money signals, DEX aggregation, and address analysis
