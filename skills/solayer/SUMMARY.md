# solayer
Solayer liquid restaking on Solana — stake SOL to receive sSOL and earn restaking rewards.

## Highlights
- Liquid restaking protocol for Solana with sSOL tokens
- Real-time APY tracking and exchange rate monitoring
- Seamless SOL staking with Jupiter DEX integration
- Balance checking for sSOL positions and SOL equivalent values
- TVL and holder statistics for protocol insights
- Secure transaction routing through onchainos CLI
- Preview mode for all write operations before confirmation
- Support for Solana mainnet with key contract addresses
