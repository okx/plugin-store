# four-meme
Buy and sell meme tokens on Four.meme bonding curve launchpad on BNB Chain.

## Highlights
- Trade meme tokens on Four.meme bonding curve launchpad
- Buy tokens with BNB from bonding curves
- Sell tokens back to bonding curves for BNB
- Get real-time token prices and market cap data
- Preview transactions before broadcasting on-chain
- Automatic token approval handling for sells
- Support for custom slippage settings
- Built-in protection with gwei-aligned amounts
