# instadapp
Instadapp Lite Vaults — deposit ETH, withdraw, and track yield on Ethereum via iETH and iETHv2 vaults.

## Highlights
- Deposit ETH into iETH v1 vault or stETH into iETHv2 vault for yield generation
- Track positions and holdings across Instadapp Lite vaults
- View real-time exchange rates and cumulative yield percentages
- Withdraw from vaults by redeeming iETH/iETHv2 shares
- Support for both native ETH deposits and ERC-4626 stETH deposits
- Leveraged stETH/WETH yield strategies across Aave, Compound, Spark, and Fluid
- Dry-run simulation for all transactions before execution
- Direct integration with onchainos wallet for secure transaction execution
