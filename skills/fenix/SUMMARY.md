# fenix
Fenix Finance plugin for swapping tokens and managing concentrated liquidity on Blast using Algebra V4 MetaDEX.

## Highlights
- Swap tokens on Fenix Finance DEX on Blast network
- Get real-time swap quotes and price information
- Add concentrated liquidity positions with custom tick ranges
- Query and browse available liquidity pools sorted by TVL
- Built on Algebra V4 MetaDEX with CLMM support
- Supports major Blast tokens (WETH, USDB, FNX)
- Dry-run mode for previewing transactions before execution
- Automatic allowance management for seamless trading
