# lido-v2
Plugin for Lido liquid staking operations including ETH staking, stETH/wstETH wrapping, and withdrawal management.

## Highlights
- Stake ETH to receive liquid stETH tokens on Ethereum mainnet
- Wrap stETH to non-rebasing wstETH for DeFi integrations
- Unwrap wstETH back to stETH across multiple chains (Ethereum, Arbitrum, Base, Optimism)
- Request ETH withdrawals with automatic splitting for large amounts
- Claim finalized withdrawal requests to receive ETH back
- Query real-time staking APR and position balances
- Check withdrawal request status and readiness
- Multi-chain wstETH support for L2 ecosystems
