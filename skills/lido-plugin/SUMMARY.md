# lido-plugin
A plugin to stake ETH with Lido liquid staking protocol, manage withdrawals, and track staking rewards on Ethereum mainnet.

## Highlights
- Stake ETH to receive liquid stETH tokens with automatic reward compounding
- Real-time APY tracking with historical trends via DeFiLlama integration
- Check stETH balances with precise share calculations
- Request ETH withdrawals from stETH with wait time estimates
- Monitor withdrawal status across pending and claimable requests
- Claim finalized withdrawals with automated checkpoint hint resolution
- Secure transaction preview and confirmation workflow for all operations
- Full integration with onchainos CLI for wallet management and signing
