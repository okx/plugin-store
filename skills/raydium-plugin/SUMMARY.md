# raydium-plugin
A Solana DEX plugin for token swaps, price queries, and pool information via Raydium AMM.

## Highlights
- Execute token swaps on Raydium DEX with customizable slippage
- Get real-time swap quotes and price impact calculations
- Query USD prices for multiple tokens simultaneously
- Access detailed pool information and liquidity data
- Browse paginated pool listings with sorting options
- Built-in safety guards for price impact and balance protection
- Seamless integration with onchainos wallet infrastructure
- Support for both dry-run previews and live transactions
