# orca
Concentrated liquidity AMM on Solana for token swaps and pool queries via the Whirlpools CLMM program.

## Highlights
- Direct integration with Orca's Whirlpools concentrated liquidity AMM
- Query pools by token pairs with TVL and volume metrics
- Get accurate swap quotes with price impact calculations
- Execute token swaps with built-in security checks
- Support for native SOL and all SPL tokens
- Automatic slippage protection and price impact warnings
- Real-time pool data from Orca's REST API
- Integrated with onchainos for secure transaction execution
