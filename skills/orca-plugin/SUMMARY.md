# orca-plugin
Concentrated liquidity AMM on Solana for token swaps and pool queries via the Orca Whirlpools program.

## Highlights
- Swap tokens on Solana's premier concentrated liquidity DEX
- Query Whirlpool pools with TVL and fee data
- Get real-time swap quotes with price impact analysis
- Built-in security scanning for output tokens
- Automatic balance verification before swaps
- Support for native SOL and all SPL tokens
- Direct integration with Orca's REST API for read operations
- Seamless execution via onchainos for write operations
