# orca-plugin
Concentrated liquidity AMM on Solana — swap tokens and query pools via the Whirlpools CLMM program.

## Highlights
- Execute token swaps on Orca's concentrated liquidity AMM
- Query Whirlpool pools with TVL and APR data
- Get real-time swap quotes with price impact analysis
- Built-in security scanning for output tokens
- Automatic balance verification before swaps
- Price impact protection (warns >2%, blocks >10%)
- Support for native SOL and SPL tokens
- Direct integration with Orca's REST API
