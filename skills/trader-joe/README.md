# trader-joe

Trader Joe Liquidity Book DEX plugin for onchainos.

Supports swap, quote, and pool discovery on Arbitrum (42161) using the LB V2.1/V2.2 protocol.

## Commands

- `quote` — Get best swap quote from LBQuoter
- `pools` — List all LB pools for a token pair
- `swap` — Execute a token swap via LBRouter
