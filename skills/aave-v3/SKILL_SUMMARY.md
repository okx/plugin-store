
# aave-v3 -- Skill Summary

## Overview
This skill enables users to interact with Aave V3, the leading decentralized lending protocol, allowing them to supply crypto assets to earn interest, borrow against collateral, and manage their DeFi positions. It provides comprehensive functionality including health factor monitoring, collateral management, efficiency mode settings, and reward claiming across Ethereum, Polygon, Arbitrum, and Base networks.

## Usage
Install the plugin and connect your wallet with `onchainos wallet login`, then use natural language commands like "supply 1000 USDC to Aave" or "check my Aave health factor" to interact with the protocol.

## Commands
| Command | Description |
|---------|-------------|
| `aave-v3 supply` | Deposit assets to earn interest |
| `aave-v3 withdraw` | Redeem supplied assets |
| `aave-v3 borrow` | Borrow against collateral |
| `aave-v3 repay` | Repay borrowed debt |
| `aave-v3 health-factor` | Check liquidation risk |
| `aave-v3 positions` | View current positions |
| `aave-v3 reserves` | List market rates and APYs |
| `aave-v3 set-collateral` | Enable/disable asset collateral |
| `aave-v3 set-emode` | Configure efficiency mode |
| `aave-v3 claim-rewards` | Collect accrued rewards |

## Triggers
Activate when users mention Aave operations like "supply to aave", "borrow from aave", "aave health factor", "repay aave loan", or want to earn yield on crypto assets through decentralized lending protocols.
