# aave-v3
Lend and borrow crypto assets on Aave V3 — the leading decentralized liquidity protocol with comprehensive DeFi operations.

## Highlights
- Supply assets to earn competitive yield rates across multiple chains
- Borrow against collateral with variable interest rates
- Real-time health factor monitoring to prevent liquidations
- Multi-chain support (Ethereum, Polygon, Arbitrum, Base)
- Collateral management with enable/disable functionality
- E-Mode efficiency for correlated assets (stablecoins, ETH derivatives)
- Automated reward claiming system
- Complete position overview with USD valuations
