# aerodrome-amm
Swap tokens and manage classic AMM (volatile/stable) LP positions on Aerodrome Finance on Base.

## Highlights
- Trade on Aerodrome's classic AMM pools (volatile and stable) on Base chain
- Get real-time swap quotes with automatic pool type detection
- Add and remove liquidity to earn trading fees from ERC-20 LP tokens
- Claim AERO token rewards from gauge emissions
- Query pool reserves and LP positions across multiple token pairs
- Auto-approve tokens and handle slippage protection
- Support for major Base tokens (WETH, USDC, AERO, CBBTC, DAI, USDT)
- Secure transaction confirmation required for all write operations
