# etherfi
Liquid restaking on Ethereum that allows depositing ETH to receive eETH, wrapping eETH to weETH (ERC-4626), and checking positions with APY.

## Highlights
- Deposit ETH into ether.fi LiquidityPool to receive eETH liquid staking tokens
- Wrap eETH into weETH (ERC-4626 yield-bearing token) for auto-compounding rewards
- Earn combined Ethereum staking + EigenLayer restaking APY
- Check balances and view current protocol APY with read-only position queries
- Preview all transactions before confirmation with two-step security gates
- Direct integration with onchainos CLI for secure transaction broadcasting
- Support for unwrapping weETH back to eETH to realize gains
- Real-time conversion rates and protocol statistics from ether.fi API
