# hyperliquid-plugin
Trade perpetuals on Hyperliquid with full position management, market/limit orders, stop-loss/take-profit brackets, and USDC deposits.

## Highlights
- Place market and limit orders with optional stop-loss and take-profit brackets in one command
- Check open perpetual positions with real-time PnL and liquidation prices
- Close positions with automatic size and direction detection
- Set stop-loss and take-profit levels on existing positions
- Get real-time market prices for all Hyperliquid perpetual markets
- Cancel open orders by ID
- Transfer funds between perpetual and spot accounts
- Deposit USDC from Arbitrum to Hyperliquid L1
