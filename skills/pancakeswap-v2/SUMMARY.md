# pancakeswap-v2
Swap tokens and manage liquidity on PancakeSwap V2 (xyk AMM) across BSC, Base, and Arbitrum networks.

## Highlights
- Token swaps on PancakeSwap V2 with slippage protection and real-time quotes
- Add and remove liquidity positions with automatic approval handling
- Multi-chain support for BSC, Base, and Arbitrum One networks
- Built-in token symbol resolution for major tokens (CAKE, USDT, USDC, WBNB)
- Dry-run mode for transaction preview before execution
- Direct integration with onchainos wallet for secure transaction signing
- Pool reserves and LP balance checking capabilities
- Constant product (xyk) AMM formula with 0.25% swap fees
