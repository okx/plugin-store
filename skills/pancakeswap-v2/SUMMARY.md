# pancakeswap-v2
Swap tokens and manage liquidity on PancakeSwap V2 (xyk AMM) across BSC, Base, and Arbitrum networks.

## Highlights
- Multi-chain support: BSC, Base, and Arbitrum One
- Token swapping with configurable slippage protection
- Liquidity provision and removal with LP token management
- Real-time price quotes and pool reserve data
- Exact-amount ERC-20 approvals for security
- Human-readable decimal input for all amounts
- Dry-run mode for transaction preview
- Built-in overflow protection for large pools
