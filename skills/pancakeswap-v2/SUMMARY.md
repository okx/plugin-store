# pancakeswap-v2
Swap tokens and manage liquidity on PancakeSwap V2 (xyk AMM) across BSC, Base, and Arbitrum networks.

## Highlights
- Multi-chain support for BSC, Base, and Arbitrum One networks
- Token swapping with customizable slippage protection and deadlines
- Liquidity provision and removal with LP token management
- Real-time price quotes and pool reserve information
- Built-in ERC-20 approval handling for seamless transactions
- Dry-run mode for transaction preview before execution
- Native token support (BNB/ETH) alongside ERC-20 tokens
- Integration with onchainos wallet for secure transaction signing
