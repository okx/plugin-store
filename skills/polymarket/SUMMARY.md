# polymarket
Trade prediction markets on Polymarket — buy and sell YES/NO outcome tokens on Polygon.

## Highlights
- Browse and search active prediction markets across politics, sports, crypto, and more
- Buy and sell YES/NO outcome tokens with USDC.e on Polygon
- Check positions, PnL, and portfolio performance across all markets
- Place limit orders or market orders with automatic slippage protection
- Cancel individual orders or all orders for a market
- Auto-derived API credentials from onchainos wallet — no manual setup required
- Support for both binary (YES/NO) and categorical (multiple outcome) markets
- Gasless order settlement via Polymarket's CLOB operator
