# fenix-finance
Fenix Finance V3 DEX on Blast for swapping, adding/removing liquidity, and querying LP positions using Algebra Integral V1 concentrated liquidity.

## Highlights
- Swap tokens on Blast network via Fenix DEX using Algebra AMM
- Add concentrated liquidity positions with custom tick ranges
- Remove liquidity and collect accrued fees from LP positions
- Query real-time swap prices and quotes through QuoterV2
- List all LP NFT positions for any wallet address
- Support for major Blast tokens (WETH, USDB, BLAST, FNX)
- Automatic token approval handling for seamless transactions
- Dry-run mode for previewing transactions before execution
